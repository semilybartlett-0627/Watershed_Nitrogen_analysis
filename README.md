# Watershed_Nitrogen_analysis
## Project Overview
This project analyzes watershed nutrient monitoring data to examine patterns in nitrogen concentrations across multiple waterbodies and sampling periods. Nutrient monitoring is an important component of water quality management because elevated nitrogen levels can contribute to harmful algal blooms in aquatic ecosystems.
## Objectives
The goal of this project was to:

* Clean and prepare watershed monitoring data for analysis
* Reshape the dataset from wide format to long format for easier analysis
* Evaluate trends in nitrogen concentrations over time
## Dataset
The dataset was too big to upload, so just a version of the cleaned data from the process was uploaded.
